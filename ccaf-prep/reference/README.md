# reference/ — bring your own (git-ignored)

This folder holds **Anthropic's official material**, which is **not redistributed** in this
repo. Everything here is git-ignored except this README, so the folder stays present after a
clone.

To get the full study experience and let the notebook-generator skill regenerate domains,
download the official guide from Anthropic and save its text here:

- **`exam_guide.txt`** — the full text of the *Claude Certified Architect – Foundations
  Certification Exam Guide* (download the PDF from Anthropic and export/copy its text).
  The [`ccaf-notebook` skill](../../.claude/skills/ccaf-notebook/SKILL.md) reads this file as
  its source of truth (verbatim task statements, the 12 sample questions).

You can also drop the Anthropic Academy **`course_catalog.txt`** here if you find it useful.
